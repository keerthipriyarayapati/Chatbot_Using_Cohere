# Chatbot_Using_Cohere

This project is a conversational chatbot built using **LangChain, Cohere, and Streamlit**.

## Features
- Interactive chatbot interface using Streamlit
- Uses Cohere LLM for generating responses
- Maintains conversation history
- Simple and user-friendly UI

## Technologies Used
- Python
- LangChain
- Cohere API
- Streamlit
- dotenv

## Installation

1. Clone the repository

```git clone https://github.com/keerthipriyarayapati/conversational-qa-chatbot.git```

2. Navigate to the project folder

```cd conversational-qa-chatbot```

3. Install dependencies

```pip install -r requirements.txt```

4. Add your Cohere API key in `.env`

```COHERE_API_KEY=your_api_key_here```

5. Run the application

```streamlit run app.py```

## Output
The chatbot allows users to ask questions and get conversational AI responses.
